git培训：
命令：
1.初始化：git init
2.添加追踪文件：git add .
3.提交版本：git commit -m"备注"
4.添加远程仓库的源：git remote add origin (地址)
5.第一次推送到远程仓库：git push -u origin master
6.之后推送远程仓库：git push
7.创建分支：git branch (分支名称)
8.转换分支：git checkout (切换的分支名称)
9.创建并转换分支：git branch -b(分支名)
10.删除分支：git branch -d(分支名称)
11.合并分支：git merge (合并分支)
12.回退版本：git reset --hard (版本号)
13.查看版本信息：git log
14.回退记录：git reflog
15.仓库状态：git status
16.克隆远程仓库：git clone (地址)
